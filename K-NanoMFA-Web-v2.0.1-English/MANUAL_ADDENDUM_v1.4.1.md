# K-NanoMFA v1.4.1 Manual Addendum

Version 1.4.1 is a maintenance release that preserves all v1.4 functions and corrects the responsive layout of the custom country and geographic-domain manager.

## Corrected interface behavior

- The country identity and provenance fields now wrap as a 2-column grid on wider screens and a 1-column grid on narrow screens.
- The domain editor grid now contracts progressively without clipping the right-hand panel.
- Long notes, URLs, and buttons wrap within the panel instead of extending beyond it.
- Country end-of-life and sewage-sludge route tables now remain within the panel width across narrower displays.

## Scientific scope

No scientific equations, parameter definitions, data presets, product-informed input calculations, uncertainty propagation, custom-geography semantics, or export logic were changed in v1.4.1. This release only corrects the interface layout and preserves v1.4 behavior.
